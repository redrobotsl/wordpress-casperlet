== Installation ==

1. Go to `Plugins` in the Admin menu
2. Click on the button `Add new`
3. Search for `Rental Directory for CasperLet` and click 'Install Now' or click on the `upload` link to upload `rental-directory-for-casperlet.zip`
4. Click on `Activate plugin`
